# Manufacturing Quality Analytics Dashboard

## Project Overview
Built an end-to-end manufacturing analytics solution to monitor machine quality, failure patterns, and operational performance using Python, PostgreSQL, and Power BI.

## Business Problem
Manufacturing teams need to identify machine failures, understand root causes, and improve production quality through data-driven decisions.

## Tools Used
- Python (Pandas, NumPy)
- PostgreSQL
- Power BI
- Excel

## Data Processing
- Cleaned raw manufacturing dataset using Python
- Removed duplicates and checked missing values
- Created calculated fields:
  - Temperature Difference
  - Failure Status

## SQL Analysis
Performed:
- Failure rate analysis
- Machine failure analysis
- Root cause analysis
- High-risk machine identification

## Dashboard Features

### Executive Overview
- Total Machines
- Total Failures
- Failure Rate
- Average Tool Wear

### Quality Analysis
- Failure trends
- Failure by machine type
- Root cause analysis

### Machine Health
- Tool wear analysis
- Torque vs failure analysis
- Machine performance monitoring

## Key Insights
- Identified machines with higher failure probability
- Analyzed factors contributing to failures
- Provided insights for preventive maintenance decisions
